（1）快速生成一个vue项目架子（webpack)  http://www.cnblogs.com/wx1993/p/6136892.html
环境要求：  安装有 Node.js、 vue、 vue-cli 。

创建项目：

vue init webpack projectName

进入项目，下载依赖：

npm install 或者 cnpm install

运行项目：

npm run dev

效果如下：
